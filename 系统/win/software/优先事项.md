## 优先事项

- 新电脑首先下载代理软件，卸载掉系统自带的终端（不然遇到win store有问题都没法更新），安装vscode并稍后同步。

- 新电脑禁止powershell运行脚本，在管理员权限的powershell里执行以下命令解除：

  ```
  Set-ExecutionPolicy -ExecutionPolicy Restricted -Scope LocalMachine
  ```

- 把桌面，文档，下载文件夹移动到d盘：

  - 在d盘新建Desktop文件夹，右键原桌面图标选择“属性” -> “位置”选项卡点击“移动” -> 选择D盘新文件夹并确定，在提示迁移文件时选择“是”即可。

  - Downloads和Documents文件夹也做类似处理。

- nvda控制面板：显示 - 更改分辨率 - 输出颜色深度改成10bpc。

- 禁用onedrive，非常不好用。

- 重装系统后可以去微软官网下载之前自带的office。微软账号使用win hello也需要在官网重新授权。

- 去掉自动管理应用的颜色（待补充）

- 桌面右键开启自动排列图标


#### 前置设置

一些优先安装的软件（通常是常驻后台的）：

1. mouseinc（在小米云盘里）

2. [火绒](https://www.huorong.cn/)

3. autohotkey：

   - 使用如下设置，并将快捷方式放到Windows启动目录：

     [AHK启动脚本](ahk.md)，可以禁用搜狗输入法的全部快捷键：ctrl+句号等

4. 搜狗输入法（语音输入总是出bug，这一部分换用讯飞）

   - 只开启系统功能快捷键-英文输入法(ctrl+shift+i)
   - 调整字号（24px）适应高分屏
   - 设置-高级-关闭桌面右下角推荐
   - 右键任务栏的 “中” 字：隐藏状态栏

5. everything：

   - 安装时选中开机自启动，everything服务，集成到右键菜单

   - everything->工具->选项：

     - 界面->选中单击托盘图标打开搜索窗口，自动聚焦于输入框
       - 结果->选中搜索关键词为空时不显示搜索结果，双击路径列打开目录
       - 视图->选中高亮光标经过行，显示大小，选中数量等等
       - 快捷键->显示窗口快捷键（Alt+F）
       - 索引->选中索引文件夹大小

6. bandizip：取消勾选解压/压缩完成后不要关闭进度窗口

7. 微信（不得不装）

8. [LenovoLegionToolkit](https://github.com/BartoszCichecki/LenovoLegionToolkit)：替换联想笔记本那个管家，开启独显直连等功能

9. steam：优先下载以使用displayfusion和wallpaper engine

   - displayfusion：用鼠标中键来转移窗口；取消窗口上的小按钮（标题栏按钮）

10. directory opus：之前买的12.33版本好像可以随便用。13版本以上就先不买了（**配置待更新**）

11. chrome：开启同步

    - fvd speed dial：手动导入小米云盘里的配置文件

    - 油猴：管理面板里导入google drive的同步文件

12. [StartAllBack](https://www.startallback.com/)：付费软件，重装系统之后可以先安装试用版等30天激活过期之后再重新使用原激活码

    - 在win任务栏设置中选中 `在所有显示器上显示任务栏 - 打开了窗口的任务栏`

13. typora：[typora配置](../../general%20tools/typora.md)

14. watt-toolkit：可以下载windows store版

15. [UACWhitelistTool](https://github.com/XIU2/UACWhitelistTool)：uac白名单小程序，生成一个不启动uac的快捷方式，可以添加到右键菜单。建议白名单的程序有：

    - starward

16. translucenttb：可以下载windows store版

    - 配合wallpaper engine一起用的话，将windows主题设置为深色，应用主题设置为浅色（**个性化 - 颜色 - 选择模式 - 自定义**）

17. sharex

18. [trafficmonitor](https://github.com/zhongyang219/TrafficMonitor/releases)：勾选 任务栏窗口设置 - 背景透明（这个透明效果跟translucenttb不太一样，待更新）
