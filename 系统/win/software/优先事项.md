## 优先事项

- 新电脑首先下载代理软件，卸载自带的windows-terminal（改用scoop管理），安装vscode并稍后同步。

  - vscode设置microsoft-authentication为msal-no-broker防止登录失败的情况，不过这个设置项应该也会同步，稍后观察一下。

- 同步一下系统时间，区域设置为美国，夜间模式强度降低到35%。

- 新电脑禁止powershell运行脚本，在管理员权限的powershell里执行以下命令解除：

  ```
  Set-ExecutionPolicy -ExecutionPolicy Restricted -Scope LocalMachine
  ```

- 把桌面，文档，下载文件夹移动到d盘：

  - 在d盘新建Desktop文件夹，右键原桌面图标选择“属性” -> “位置”选项卡点击“移动” -> 选择D盘新文件夹并确定，在提示迁移文件时选择“是”即可。

  - Downloads和Documents文件夹也做类似处理。

- nvidia 控制面板：显示 - 更改分辨率 - 输出颜色深度改成10bpc。重装系统后nvidia  app也要重新下。

  - 有时候nv托盘图标会消失，可以通过右键单击桌面打开Nvidia控制面板。点击顶部的“桌面”选项卡，取消勾选“显示通知托盘图标”，然后再重新勾选一次。

- 禁用onedrive，非常不好用。

- 重装系统后可以去微软官网下载之前自带的office。微软账号使用win hello也需要在官网重新授权。

- 显示器没有颜色配置时，去掉自动管理应用的颜色（系统 - 屏幕 - 颜色配置文件）。2k屏200%缩放，4k屏300%缩放。

- 桌面右键开启自动排列图标。

- 右下角不显示日历：是被系统隐藏了，点击一个下拉小箭头可以看到，详见[link](https://zhuanlan.zhihu.com/p/707350207)。有一说一挺无语的。

- 搜狗输入法（语音输入总是出bug，这一部分换用讯飞）

   - 设置 - 高级 - 关闭桌面右下角推荐

#### 可以用scoop的软件

- 这些软件可以用scoop装，区别于开发工具，使用频率也不高

- 优先级比较高的（用管理员模式启动win-terminal，因为icaros需要这个）：

  ```
  si pdf-xchange-editor telegram discord imageglass ffmpeg screentogif neteasemusic qqmusic mpv yt-dlp icaros-np dismplusplus hasher renamer locale-emulator recuva feem qalculate powertoys
  ```

  - pdf-xchange-editor：

    - 勾选文档 - 打开上次的文档；页面显示 - 默认缩放 - 适合可见；页面显示 - 默认页面设置 - 连续；
    - 设置 Esc 键为选择文本快捷键（右键工具栏 - 自定义工具栏 - 命令 - 搜索”选择文本“）
  - mpv：

    - [让B站视频可以在mpv中播放](https://github.com/diannaojiang/Bilibili-Playin-Mpv)，导入[mpv.reg](resources/mpv.reg)
    - ~~进阶设置~~：https://bbs.acgrip.com/thread-7443-1-1.html
  - 软件作用：
    - yt-dlp：下载视频的，比如YouTube
    - icaros：为视频格式提供缩略图预览，安装后只需要打开一次
    - recuva：数据恢复

- （可选）优先级一般的：

  ```
  si qbittorrent-enhanced firefox magpie qtscrcpy fileactivitywatch screenoff extras/kodi twinkle-tray obs-studio mkvtoolnix rufus zoom
  ```

  - qbittorrent-enhanced（自从用了网盘，没用过这个）：

    - 行为：设置启动时最小化，最小化到系统托盘
    - Bittorrent：设置做种限制
  - 软件作用：
    - magpie：放大镜
    - qtscrcpy：安卓在win上运行，但是效果远不如各家自己的软件，比如小米电脑管家
    - rufus：跟微pe工具箱差不多的功能，国内估计用得少
